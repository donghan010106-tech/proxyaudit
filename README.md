# ProxyAudit

ProxyAudit is an interactive console for fair AI-based resume screening. It
localizes the modality through which a protected attribute leaks into a
hiring model, issues a per-decision certificate of whether that leak was the
actual cause of a given outcome (necessity and sufficiency), and drives a
targeted, certificate-guided repair rather than a blanket retraining. The
console reports repair strategies against measured fairness/utility
trade-offs, a residual-leakage probe, and a live recovery feed with
per-decision recommendations and an auditor chat grounded in the certified
cases.
